- Frontend: Vue
- Backend: AWS Lambda
- Hosting: Cloudflare, Fly.io
- Infrastructure: AWS SAM
- LLM: OpenAI API
- Vector Search DB: Qdrant
- Local Development: Docker, LocalStack
- Others: OpenStreetMap, Mapbox, Maplibre
